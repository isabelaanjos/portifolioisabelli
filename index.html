Palavra aleatória

Este projeto exibe uma palavra aleatória na tela, mas ela é revelada gradualmente conforme o movimento do mouse. O código começa com a escolha de uma palavra aleatória de uma lista (como "caminhante", "caminho" ou "caminha"). A função palavraParcial controla a quantidade de letras da palavra que são exibidas com base na posição horizontal do mouse (mouseX). A cada movimento do mouse da esquerda para a direita, mais letras da palavra são reveladas, criando um efeito de "digitação" ou revelação progressiva. O fundo da tela é definido como branco e o texto é centralizado e exibido em preto, com um tamanho grande (64px). O código utiliza a função map para ajustar o número de letras exibidas dependendo da posição do mouse. O resultado é uma interação simples onde a palavra vai se formando à medida que o usuário move o mouse.

Lupa esquisita

Este projeto cria uma interação visual em que o objetivo é encontrar um ponto aleatório na tela. O ponto é gerado inicialmente em uma posição aleatória dentro de uma área de 400x400 pixels e, a cada quadro, o ponto se move suavemente de maneira aleatória. A movimentação do ponto é controlada pelas funções random(-5, 5), que ajustam sua posição nas direções X e Y, e o constrain garante que o ponto permaneça dentro dos limites da tela (0 a 400 pixels em ambas as direções). Enquanto isso, o código desenha um círculo na posição do mouse, e o tamanho desse círculo varia com base na distância entre o mouse e o ponto aleatório gerado. A cada movimento do mouse, o círculo aumenta ou diminui conforme a proximidade entre a posição do mouse e o ponto, criando uma sensação de "caça" ao ponto. Quando a distância entre o mouse e o ponto se torna menor que 3 pixels, o código exibe a mensagem "Encontrei!" no centro da tela, e a função noLoop() é chamada para parar a animação, ou seja, o movimento do ponto é interrompido e o jogo/experiência chega ao fim. Em resumo, este projeto cria uma dinâmica simples de busca, onde o usuário tenta encontrar o ponto gerado aleatoriamente, interagindo com a tela ao mover o mouse. A animação é interrompida assim que o ponto é encontrado, proporcionando um desafio visual rápido e envolvente.

Palavra aleatória

Este projeto cria uma interação visual onde uma palavra aleatória é revelada progressivamente conforme o movimento do mouse. O funcionamento básico é o seguinte: Escolha Aleatória da Palavra: Ao iniciar o programa, uma palavra é escolhida aleatoriamente de um conjunto de palavras pré-definidas (como "caminhante", "caminho" ou "caminha"). A função palavraAleatoria é responsável por fazer essa seleção. Revelação da Palavra: A função palavraParcial controla quantas letras da palavra são exibidas, dependendo da posição do mouse na tela. Ela usa a função map para mapear a posição horizontal do mouse (mouseX) para a quantidade de letras que devem ser mostradas. O valor de mouseX é usado para determinar o número de letras visíveis, de forma que, à medida que o mouse se move da esquerda para a direita (de 0 a 400 pixels), mais letras da palavra são exibidas. Função de Desenho: Dentro da função draw, que é chamada continuamente pelo p5.js, o fundo da tela é sempre reiniciado para branco com a função inicializaCores. Em seguida, a palavra parcial (ou seja, a parte da palavra que deve ser mostrada) é desenhada na tela com a função text. O texto é centralizado na tela com a posição 200, 200 e exibido com uma fonte grande de 64px. Interação Visual: O resultado é um efeito visual interessante em que, dependendo da posição do mouse, o usuário vê uma "revelação" gradual da palavra. Quando o mouse está à esquerda, apenas a primeira letra da palavra é visível. À medida que o mouse se move para a direita, mais letras vão aparecendo até que a palavra completa seja revelada. O código usa conceitos básicos do p5.js, como createCanvas, text, textAlign, e manipulação de cores e fontes. A interação com o movimento do mouse cria uma experiência dinâmica e simples, tornando o projeto ideal para demonstrar como eventos de entrada do usuário podem ser usados para controlar o comportamento visual em tempo real.
